  • XTech 2006

    XTech 2006 is is coming up, and I’ll be speaking about the state of web graphics, specifically about incorporating graphics in web applications. There will be some SVG, and some Canvas; some 2D and some 3D. Hopefully there will be a handful of cool tech demos, as well as some examples of real-world apps that are using these technologies.

    I’ll post the presentation here after it’s been given — the main message I want to get across is that web apps can progress beyond form input and text/numeric data display without having to resort to Flash or other non-web methods. We’ll see how that goes…

    Other than myself, there are going to be a whole bunch of Mozilla folks in attendance and presenting, so it should be a good conference.
